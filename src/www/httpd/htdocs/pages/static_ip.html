Configurations

In this page you can configure a static IP address.
If you don't configure IP address and subnet mask here, dhcp will be used.
Please note: if you enter the wrong IP address you will no longer be able to access your cam.

Static IP

IP address Enter your IP address.
Subnet mask Enter the subnet mask.
Gateway Enter the IP address of your gateway or leave blank.
DNS 1 Enter the IP address of your DNS server.
DNS 2 Enter the IP address of your DNS server.
Any change will take effect after the reboot of the camera.
